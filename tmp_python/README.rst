Basic setup
--------------
#. Have npm installed
#. Run :code:`npm install` to install all needed dependencies.
#. Run :code:`npm run` to start the dev-server.
#. Navigate to `localhost:1234/index.html`__ to play the game.

__ localhost:1234/index.html
